Authorization Server in Spring Boot 3.1

Jul 04, 2023 DaShaun Carter

The site source code is available on https://github.com/dashaun-dev.

Today, there is a Spring Cloud Gateway deployed, and it sits in front of this JBake website.

Today, I added Spring Authorization Server and just setup an InMemoryUserDetailsManager.

This setup is very similar to what Josh Long and Steve Riesenberg did in their video Going Full OAuth with the new Spring Authorization Server in Spring Boot 3.1. I just took it a little further.

  • I made everything publicly accessible by using a Cloudflare Argo Tunnel

  • I pushed it all to production

At the moment (July 4th, 2023), I have these services deployed to KNative 1.10, on Docker Desktop, running on my Mac dev machine.
